# mentor.skill

蒸馏导师的指导风格、职业建议和领导智慧到一个 AI Skill 中。

## 使用方法

| 命令 | 说明 |
|------|------|
| `/mentor` | 调用导师 Skill 创建器 |

## 功能说明

Mentor.skill 是一个 Claude Code 扩展，用于将导师的指导风格、职业建议和领导智慧蒸馏成一个 AI Skill。

### 工作流程

1. **录入信息**：称呼、基本信息、指导风格
2. **导入原材料**：工作消息、邮件、会议纪要等
3. **分析生成**：提取指导方式、决策风格、标志性元素
4. **确认写入**：生成 Mentor Skill 文件

### 原材料类型

- 工作消息记录
- 邮件往来
- 1on1 会议纪要
- 绩效反馈
- 直接粘贴文字
