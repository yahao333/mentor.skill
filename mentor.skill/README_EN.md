# Mentor.skill

Distill a mentor's guidance style, career advice, and leadership wisdom into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/mentor` | Invoke the Mentor Skill creator |

## Description

Mentor.skill is a Claude Code extension that distills a mentor's guidance style, career advice, and leadership wisdom into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, guidance style
2. **Import Materials**: Work messages, emails, meeting notes, etc.
3. **Analyze & Generate**: Extract guidance style, decision-making traits, iconic elements
4. **Confirm & Write**: Generate Mentor Skill file

### Material Types

- Work chat records
- Email exchanges
- 1on1 meeting notes
- Performance feedback
- Paste text directly
